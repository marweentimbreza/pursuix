GitLab push

Lets "publish → gitlab" commit straight to your repo. Stored in this browser only.

A token in browser storage is a real key. Use a project-scoped token with only write_repository, on your own machine. Anyone with this browser can read it.
Default gitlab.com. Self-hosted? Put its base URL.
Project → Settings → General shows the numeric Project ID (most reliable).
Project → Settings → Access Tokens → role Developer, scope write_repository.
